# RAG Chatbot LLM

## Overview
An AI-powered Retrieval Augmented Generation (RAG) chatbot that combines Large Language Models with document retrieval and semantic search to provide accurate responses.

## Features
- Document Retrieval
- Semantic Search
- Context-Aware Responses
- LLM Integration
- Vector Database Support

## Technologies Used
- Python
- LangChain
- FAISS
- OpenAI
- Streamlit

## Installation

pip install -r requirements.txt

## Run

python app.py
